MapThing

A library by Jon Reades for the Processing programming environment.
Last update, 04/04/2013.

A collection of classes for reading and displaying Shape files (a.k.a. ESRI shapefiles), CSV, and GPX data in a map-based sketch.
There are four main classes with which you should be familiar: 1) BoundingBox defines the sketch's geographic 'envelope' by mapping a shapefile's geographic coordinates on to the viewable space of the sketch; 2) Lines is used to read and display single-part and multi-part line data; 3) Points is used to read and display point data; and 4) Polygons is used to read and display single and multi-part polygon data (this only works with shapefiles for now). GPX display needs work.

Download

Download MapThing version 1.4.1 (1.5) in .zip format.

Installation

Unzip and put the extracted MapThing folder into the libraries folder of your Processing sketches. Reference and examples are included in the MapThing folder.

Keywords. esri,shape,geography,projection,polygon,point,line,csv,map,gpx
